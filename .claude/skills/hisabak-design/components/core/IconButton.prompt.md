Circular icon-only button — app bars, list rows, anywhere a labelled Button is too heavy.

```jsx
<IconButton icon="notifications" />
<IconButton icon="add" variant="accent" filled />
```

Variants: `plain` (transparent), `soft` (sunken fill), `accent` (green tint). Sizes `sm`/`md`/`lg`. Icon is a Material Symbols Rounded ligature; `filled` swaps to the filled glyph.
